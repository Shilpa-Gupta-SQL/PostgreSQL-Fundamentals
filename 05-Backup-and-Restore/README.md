# Backup and Restore in PostgreSQL

## Overview
This section explains different backup and restore strategies available in PostgreSQL.

## Topics
- Logical backups (pg_dump, pg_restore)
- Physical backups
- Full backup vs incremental concepts
- Point-in-Time Recovery (PITR)
- Importance of WAL in backups

## Goal
To understand how PostgreSQL data can be safely backed up and restored.
